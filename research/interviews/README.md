# Interviews

Working directory for the Tuesday interview-brief routine (`automations/02-interview-brief.md`).
One file per topic: `YYYY-MM-DD-<slug>.md`, containing the idea in three sentences, five
questions, and an `## Author answers` section.

For the author: answer in English, 中文, or both mixed; fragments and voice-dump quality
are exactly right. 15–30 minutes is enough. The Thursday drafter turns whatever is here
into the draft — no answers simply means that week produces a smaller piece, which is fine.
